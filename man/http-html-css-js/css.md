### CSS 有哪些可控
```text
CSS（层叠样式表）是用于控制网页或应用程序的外观和样式的语言。它提供了广泛的控制选项，用于操纵元素的外观和布局。下面列出了一些常见的可以通过 CSS 控制的方面：

元素的尺寸和布局：通过设置元素的宽度、高度、边距、内边距和定位属性，可以控制元素在页面上的尺寸和布局。

字体和文本样式：可以通过设置字体系列、字体大小、字体粗细、字体颜色、对齐方式、行高和文本装饰等属性来控制文字的样式。

背景和边框样式：可以设置元素的背景颜色、背景图像、背景重复方式以及边框样式、边框宽度和边框颜色。

元素的可见性和显示方式：可以使用可见性属性（如 display 和 visibility）来控制元素的显示与隐藏，以及使用 float 和 position 等属性来控制元素的布局方式。

元素的动画和过渡效果：通过使用 CSS 动画和过渡属性（如 animation 和 transition），可以创建元素的动态效果，例如渐变、旋转、缩放等。

响应式设计：可以使用媒体查询和响应式布局技术，在不同的屏幕尺寸和设备上调整元素的样式和布局。

列表和表格样式：可以设置列表项的样式，如项目符号、编号、间距等，以及表格的样式，如边框、背景色、对齐方式等。

悬停和交互效果：可以通过 :hover、:active 和 :focus 等伪类选择器来定义元素在鼠标悬停、被点击或获得焦点时的样式。

这只是 CSS 提供的一些常见的样式控制选项，实际上还有更多的属性和技术可以用于调整元素的外观和样式。开发人员可以根据需求和 k.要求，使用 CSS 对网页或应用程序进行高度定制的样式设计。
```
